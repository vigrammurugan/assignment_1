# Sales Intelligence Hub

## Project Overview

Sales Intelligence Hub is a powerful sales analytics and financial tracking system developed using Python, MySQL, and Streamlit. This application helps organizations efficiently manage branch-wise sales operations, payment collections, pending balances, customer records, and sales performance monitoring through an interactive dashboard.

The system is designed to simplify business decision-making by providing real-time sales insights, automated calculations, and structured database management.

---

## Key Features

* Multi-Branch Management System
* Admin & Super Admin Login Authentication
* Sales Tracking & Monitoring
* Payment Collection Management
* Pending Amount Analysis
* Customer & Product Management
* Interactive Dashboard using Streamlit
* MySQL Relational Database Integration
* SQL Queries, Triggers & Generated Columns
* CRUD Operations (Create, Read, Update, Delete)
* Real-Time Financial Reporting
* Secure Database Connectivity
